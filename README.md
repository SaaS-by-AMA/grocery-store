# Grocery Store App

A simple online grocery store where users can buy food items.

## Features
- Browse products (drinks, chips, snacks, etc.)
- Add items to cart
- Checkout and payment
- User accounts

## How to Use
1. Visit the website
2. Browse products
3. Add items to your cart
4. Checkout and pay
5. Get your order delivered
